jquery_DOM_class조작

문서객체모델(DOM) - class조작
사용자가 선택한 요소에 클래스 서식을 추가하거나 삭제 또는 클래스가 없는 요소를 찾아서 서식을 적용하는 메서드이다.

1) addClass(); - 선택한 요소에 클래스 서식을 추가한다.
2) removeClass(); - 선택한 요소에 클래스 서식을 제거한다.
3) toggleClass(); - 선택한 요소에 클래스 서식을 추가하거나 제거한다.
4) hasClass(); - 선택한 요소에 해당 클래스 이름이 이는지 여부를 판단하여 true, false값으로 표시
* css(); - 사용자가 선택한 요소에 스타일 서식을 적용한다.



제이쿼리 문서객체모델(DOM) - 내용추가, 삭제

객체 편집 메소드
html 문서안에서 사용자가 선택한 요소의 앞쪽이나 뒤쪽에 내용을 추가하거나 복제, 삭제하는 메소드

1. 내용추가하기(add)
1) append(); - 선택한 요소(부모)의 안쪽으로 맨 뒤에 내용을 추가하기
2) prepend(); - 선택한 요소(부모)의 안쪽으로 맨 앞에 내용을 추가하기
3) after(); - 선택한 요소(형제)의 뒤에 내용을 추가하기
4) before(); - 선택한 요소(형제)의 앞에 내용을 추가하기
5) insertBefore(); insertAfter(); - 자리배치하기

*A insertAfter(B), A insertBefore(B)를 사용하면 A라는 요소의 앞쪽에 B라는 요소를 배치할 수 있고, B라는 요소 뒤쪽에 A요소를 배치할 수도 있다. after(), before()메소드와 기능은 같지만 사용법이 다르다.

예)
$('A').after('B'); - A요소의 바깥쪽으로 뒤에 B내용을 추가한다.
$('B').insertAfter('A'); - A요소의 바깥쪽으로 뒤에 B내용을 추가한다.



2. 내용 삭제하기(Remove)
사용자가 선택한 요소를 찾아서 내용만 지우거나, 태그요소 자체를 삭제할 수 있다.

1) empty(); - 사용자가 선택한 요소의 내용만 삭제
2) remove(); - 사용자가 선택한 요소를 삭제

쿠키(cookie)

쿠키(cookie): 서버에서 클라이언트에게 특정 정보를 저장하는 데이터
사용자 이름, 아이디, 방뭄 횟수 등을 기록하여 차후 접속시 활용

* 제이쿼리 쿠키 작성법
$.cookie('쿠키이름', '쿠키값', {expires: 만료일, path: '저장경로'});
